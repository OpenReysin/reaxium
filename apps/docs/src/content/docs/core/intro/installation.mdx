## Package Managers

<Tabs>
    <TabItem label="npm">
        ```bash
        npm install @reaxium/core
        ```
    </TabItem>
    <TabItem label="pnpm">
        ```bash
        pnpm add @reaxium/core
        ```
    </TabItem>
    <TabItem label="yarn">
        ```bash
        yarn add @reaxium/core
        ```
    </TabItem>
    <TabItem label="bun">
        ```bash
        bun add @reaxium/core
        ```
    </TabItem>
</Tabs>

---

## CDN (Browser)

```html
<script type="module">
  import { atom, computed, watch } from 'https://unpkg.com/@reaxium/core?module';
  // Use Reaxium here
</script>
```

---

## Requirements

- Node.js 16+ (for npm/yarn/pnpm)
- Modern browser (for CDN)
